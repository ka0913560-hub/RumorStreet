# RumorStreet Audio Assets

## Background Music
Place your background music file here as `background-music.mp3`

For temporary testing, you can use any royalty-free music file or the app will work silently.

Recommended sources for futuristic game music:
- https://pixabay.com/music/
- https://freemusicarchive.org/
- https://incompetech.com/

The audio file should loop continuously across all pages.
